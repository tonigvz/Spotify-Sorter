# Spotify-Sorter

This Python script automates the creation of a Spotify playlist based on specified criteria, utilizing the Spotify Web API. It fetches track information from a predefined playlist, filters tracks by artist genre, and creates a new playlist with tracks that match the specified genre criteria.

## Features

- Fetches track information from a Spotify playlist.
- Filters tracks based on the genre of their respective artists.
- Creates a new Spotify playlist and adds filtered tracks to it.
- Utilizes concurrency for efficient API calls.

## Usage

1. Set up your Spotify API credentials and playlist information in a `.env` file.
2. Run the script to fetch track information and create a new playlist with filtered tracks.

## Dependencies

- Python 3.x
- Spotify API credentials
- dotenv
- requests

## Getting Started

1. Clone this repository.
2. Install the required dependencies using pip.
3. Configure your Spotify API credentials in a `.env` file.
4. Run the script using Python.
